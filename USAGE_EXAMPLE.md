| email | String | 是 | 邮箱格式，唯一 | 邮箱 |
| phone | String | 是 | 11位数字，唯一 | 手机号 |
| verifyCode | String | 是 | 6位数字，5分钟有效 | 验证码 |

- 规则1: 用户名唯一性校验
- 规则2: 邮箱唯一性校验
- 规则3: 手机号唯一性校验
- 规则4: 验证码5分钟内有效
- 规则5: 注册成功发送欢迎邮件

- 依赖1: 验证码服务（发送/验证）
- 依赖2: 邮件服务（欢迎邮件）
- 依赖3: 数据库（用户表）

### 1.6 风险点识别
- 风险1: 验证码绕过（安全风险）
- 风险2: 并发注册导致唯一性失效
- 风险3: 邮件服务不可用导致注册失败
