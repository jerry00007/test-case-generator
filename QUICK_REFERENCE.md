# 测试用例生成器 - 快速参考卡片

## 🚀 增强版特性

### ✅ 10大测试维度（vs. 旧版6个）
1. ✅ **功能测试** - 正常/分支流程
2. ✅ **输入测试** - 合法/非法/空值/超长/特殊字符
3. ✅ **边界测试** - 临界值/上下边界/边界外值
4. ✅ **业务规则** - 多条件组合/逻辑冲突
5. ✅ **状态流转** - 正常流转/非法跳转
6. 🆕 **异常测试** - 参数缺失/数据不存在/服务异常
7. 🆕 **并发与幂等性** - 重复请求/并发/一致性
8. 🆕 **容错与稳定性** - 超时/重试/降级/部分失败
9. 🆕 **安全测试** - SQL注入/XSS/越权/Token
10. 🆕 **性能测试** - 高并发/大数据量

### ✅ API接口测试7维度
1. ✅ 参数校验 - 必填/类型/长度/非法值
2. ✅ 功能验证 - 正常请求/业务正确性
3. ✅ 异常处理 - 错误码/异常返回
4. 🆕 **幂等性** - 重复调用结果一致
5. 🆕 **并发** - 多请求竞争
6. 🆕 **性能** - 响应时间/QPS
7. 🆕 **容错** - 超时/重试/降级

### ✅ 自检机制（5个关键问题）
1. ❓ 是否覆盖所有功能点？
2. ❓ 是否遗漏边界情况？
3. ❓ 是否覆盖异常流程？
4. ❓ 是否存在遗漏的高风险场景？
5. ❓ 是否有冗余或重复用例？
